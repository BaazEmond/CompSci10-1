All About Programming

When you bring up the topic of programming, you can't guess what aspect of programming they want you to look at. There are many different languages in programming, and I will be telling you a few of the ones that exist.

HTML

HTML, or HyperText Markup Language, is the fundamental building block of the Web and defines the content and structure of a web page. Below are some common page elements that HTML is used for:

  • Headings
  • Paragraphs
  • Images
  • Links
  • Lists

Learn more about HTML with the w3Schools HTML Tutorial.

CSS

CSS, or Cascading Style Sheets, is used to describe a web page's appearance and presentation. Below are some common uses of CSS:

  • Fonts
  • Colors
  • Layout
  • Backgrounds

Learn more about CSS with the w3Schools CSS Tutorial.

JavaScript

JS, better known as JavaScript is the programming language of the web and controls the functionality and behaviour of a web page. Below are some common uses of JavaScript:

  • Add Interactivity to a Web Page
  • Create Web Apps
  • Create Web Games & Animations

A great JavaScript reference is The Modern JavaScript Tutorial.

There are many more programming languages that exist, but these three are the ones that I know well.
